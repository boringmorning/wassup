第一頁為登入畫面 若是帳密錯誤會顯示"somegthing wrong"的警告
    若登入帳號為admin則會進入admin專用的頁面 
    按下sign up 可以進入註冊畫面進行註冊 按下forget password可以找會密碼
    
    在註冊畫面填完應填的資料後按下"create"便可成功註冊 按return 回到登入頁面
    在忘記密碼的頁面之中填完所有資料後 若id與日期是正確的話便可成功改寫密碼，否則不會發生任何事

    在一般使用者介面 可以進行 留言 與修改密碼
    在留言介面 你在上方可以看到妳的暱稱與性別
    下方的則可以輸入文字進行留言 
    清除留言可以把你剛剛輸入的文字清空
    按回上頁可以返回上一頁
    下面則顯示現在時間

    修改密碼再輸入完所有應填入的資料後按"修改密碼"便可完成修改 按"return" 返回上一頁

    在管理者介面除了留言與修改密碼外多了刪除與修改留言的功能
    在右方的輸入欄輸入你想刪除或修改的留言編號按下"刪除留言" 或是 "修改留言" 便可以對留言進行改動
